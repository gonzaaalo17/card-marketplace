About this project

Project Description – Online Trading Card Marketplace

This project is focused on developing a web application for buying and selling collectible card game cards, such as Pokémon, Yu-Gi-Oh!, Magic: The Gathering, and more. The main goal is to create a platform where users can register, list cards for sale, search for cards by different criteria, and view detailed information about each card in a simple and user-friendly way. The idea for this project comes from my studies at SUPSI (Scuola Universitaria Professionale della Svizzera Italiana) as part of the Web Applications course. The academic purpose is to learn and apply fundamental web development concepts in a practical context, starting with static HTML pages that will later be extended with dynamic functionality and backend integration. This project not only allows me to get familiar with HTML and semantic web page structure but also lays the foundation for implementing CSS and JavaScript in future phases, as well as integrating a backend and database to handle dynamic content. As a SUPSI student, this project gives me the opportunity to apply theoretical knowledge in a practical and realistic scenario while developing essential skills in modern web development.
